# Taller-backend

Primero se creo el entorno con estos comandos: 
> mkdir myproject  #este comando no se uso porque ya se crearon las carpetas previamente
> cd myproject     #este comando no se uso porque ya se crearon las carpetas previamente
> py -3 -m venv .venv 

Luego se se activa el entorno con:
> .venv\Scripts\activate

Luego se instala Flask con: 
$ pip install Flask
